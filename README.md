# Mec0-de Brew

## How do I install these formulae?

`brew install mec0-de/brew/<formula>`

Or `brew tap mec0-de/brew` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "mec0-de/brew"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
